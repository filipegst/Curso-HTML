Principais formataçoes

Negrito/ Destaque

não se deve utilizar o negrito com a tag b

em vez disso utilize a tag strong pois ela é semantica

Italico

não se deve utilizar o italico utilizando a tag i

em vez dela deve se utilizar a tag em de emphasis pois ela e semantica

texto marcado

podemos tambem criar uma marcação com o termo que deixa o texto marcado

endereço

para endereços se deve utilizar a tag

address que indica um endereço de forma semantica
